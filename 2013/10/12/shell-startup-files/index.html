Запуск shell

В разных shell оболочках есть множество файлов запускающихся сразу при запуске, чтобы установить пременные окружения и другие функции автолматизации.

Временами это вылетает из головы а лезть в man неохота? Так что это заметка скорее для себя, но если кому то она окажется полезной это хороше.

Shell бывает:

  • login - тот который прописан увас в /etc/passwd или запущен с опредиленным флагом. Он на то и loing чтобы запускаться утилитой login.
  • интерактивный - тот под которым вы работаете в интерактивном режиме.
  • обычный - это тот в котором запускаються скрипты.

shell который вы по умолчанию запускаете из иксов в виртуальном-терминале не являеться логин сшеллом, потому что вы уже авторизированы.

bash

login
yes no
interactive yes profile bashrc
no profile -

zsh

login
yes no
interactive yes zshenv zprofile zshrc zlogin zshenv zshrc
no zshenv zprofile zlogin zshenv
